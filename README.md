# Cadence

Cadence is an arcade-style typing test platform built from the exported zip design.

## Run locally

Open `index.html` in a browser.

The exported runtime in `support.js` loads React, ReactDOM, and Babel from `unpkg.com`, so an internet connection is required for the page to boot.

## Publish

This is a static site. It can be published from the repository root with GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host.
